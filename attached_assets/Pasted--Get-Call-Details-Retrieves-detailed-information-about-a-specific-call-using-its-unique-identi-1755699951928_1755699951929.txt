# Get Call Details

> Retrieves detailed information about a specific call using its unique identifier.

## OpenAPI

````yaml get /calling/call-details
paths:
  path: /calling/call-details
  method: get
  servers:
    - url: https://prod-api.ringg.ai/ca/api/v0
  request:
    security: []
    parameters:
      path: {}
      query:
        id:
          schema:
            - type: string
              required: true
              description: (Required) The unique identifier (UUID) of the call.
      header:
        X-API-KEY:
          schema:
            - type: string
              required: true
              description: (Required) Your Ringg AI API key.
      cookie: {}
    body: {}
  response:
    '200':
      application/json:
        schemaArray:
          - type: object
            properties:
              id:
                allOf:
                  - type: string
              status:
                allOf:
                  - type: string
              start_time:
                allOf:
                  - type: string
                    format: date-time
              end_time:
                allOf:
                  - type: string
                    format: date-time
              participants:
                allOf:
                  - type: array
                    items:
                      type: string
        examples:
          example:
            value:
              id: <string>
              status: <string>
              start_time: '2023-11-07T05:31:56Z'
              end_time: '2023-11-07T05:31:56Z'
              participants:
                - <string>
        description: Call details retrieved successfully.
    '404':
      application/json:
        schemaArray:
          - type: object
            properties:
              detail:
                allOf:
                  - type: string
                    example: Call not found
        examples:
          example:
            value:
              detail: Call not found
        description: Not Found - The call does not exist.
    '500':
      application/json:
        schemaArray:
          - type: object
            properties:
              detail:
                allOf:
                  - type: string
                    example: An unexpected error occurred.
        examples:
          example:
            value:
              detail: An unexpected error occurred.
        description: Internal Server Error.
  deprecated: false
  type: path
components:
  schemas: {}

````
