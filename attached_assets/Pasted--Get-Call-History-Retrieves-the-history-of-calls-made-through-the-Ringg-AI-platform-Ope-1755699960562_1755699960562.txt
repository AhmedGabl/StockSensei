# Get Call History

> Retrieves the history of calls made through the Ringg AI platform.

## OpenAPI

````yaml get /calling/history

    security: []
    parameters:
      path: {}
      query:

            - type: string

              default: 0
        agent_id:

            - type: string
              description: Filter calls by assistant ID.

            - type: string
              description: Filter calls by status (e.g., "completed", "failed").

          schema:
            - type: boolean
              description: Whether to download the results as a CSV file.
        bulk_list_id:
          schema:
            - type: string

                          format: uri

                          type: string

                          type: object
                          properties:
                            id:
                              type: string
                            agent_name:
                              type: string
                        call_duration:
                          type: number
                        call_attempt_time:
                          type: string
                          format: date-time
                        call_cost:
                          type: number
                        call_type:
                          type: string
                        inbound_from:
                          type: string
                          nullable: true
              limit:
                allOf:
                  - type: integer
              offset:

                  - type: integer

                  - type: integer

                  - type: integer
````
